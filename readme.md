**Proyecto académico desarrollado para la UTN – Facultad Regional La Plata.**

La aplicación permite registrar pacientes y llevar un control diario de las comidas
para tener un seguimiento más claro del consumo de calorías.

La idea del proyecto es que el nutricionista pueda ver rápidamente cómo viene
el día del paciente y tener un registro de las ingestas.

## Funcionalidades

- Alta y gestión de pacientes
- Registro de comidas diarias
- Cálculo de calorías totales
- Dashboard simple con resumen del día

## Tecnologías

- Python
- Django
- HTML / CSS
- Bootstrap
